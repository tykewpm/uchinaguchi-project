About This Project

The Uchinaguchi Project began as a mission to collect and preserve digital resources for learning the Okinawan language. Using automated tools and community contributions, we've gathered videos, dictionary entries, and educational materials from across the internet.

Our goal is to make Uchinaguchi learning accessible to anyone, anywhere, while supporting the global effort to revitalize this endangered language.

Methodology

  • Automated web scraping of educational resources
  • YouTube API integration for video discovery
  • Community-driven content verification
  • Open-source tools for ongoing data collection

Data Sources

  • YouTube educational channels
  • University of Hawaii resources
  • University of the Ryukyus
  • Cultural organization websites
  • Community contributions
